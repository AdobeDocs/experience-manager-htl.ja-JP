# HTL TCK{#the-htl-tck}

## HTL TCK とは

[HTML Template](https://docs.adobe.com/docs/en/htl.html "LanguageHTML(HTML)の概要（旧称Sightly）は、") Adobe Experience Manager [](http://www.adobe.com/solutions/web-experience-management.html) 6.0で導入されたもので、HTML用の推奨および推奨サーバー側テンプレートシステムとしてJSP(JavaServer Pages)に置き換えられます。

The HTML Template Language Technology Compatibility Kit - HTL TCK - provides a suite of tests meant to certify that an HTL implementation conforms with the official [language specification](https://github.com/adobe/htl-spec).
