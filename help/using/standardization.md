# 標準化 {#standardization}

HTL 標準を次に示します。

* [HTL の仕様](https://github.com/adobe/htl-spec) - 経験豊富なソフトウェア開発者向けの、形式的構文と機能説明。
* [HTL TCK](https://github.com/adobe/htl-tck) - 言語の互換性を認定するテストスイート。
