# 開発者ツール {#developer-tools}

HTL には次の開発ツールを使用できます。

* [HTL Read Eval Print Loop](https://github.com/adobe/aem-htl-repl)  — 学習に最適な、言語を使用した再生用のライブ実行環境
* [HTL Maven プラグイン](https://sling.apache.org/components/htl-maven-plugin/) - Apache Sling HTL Maven プラグインは、ビルド時に、プロジェクトからHTMLテンプレート言語スクリプトの検証をサポートします。
* [AEM Eclipse 拡張機能](https://experienceleague.adobe.com/docs/experience-manager-cloud-service/content/implementing/developer-tools/eclipse.html)  — バックエンド開発者がAEMのプロジェクトを作成および編集するためのツール
